# AURA Ring 2 Website

Premium 3D-driven wellness ring product site.
Built with Next.js 14 App Router, TypeScript, Tailwind v3, GSAP 3 + ScrollTrigger, Lenis, React Three Fiber.

## Getting Started

1. Drop the `aura-ring.glb` file into `public/models/`.
2. Add image and video assets to `public/assets/`.
3. Run `npm install --legacy-peer-deps`
4. Run `npm run dev`

## Deployment
Netlify deployment is configured via `netlify.toml`.
